# Aprendendo React Usando IA

## Introdução

Este repositório documenta meu processo de aprendizado em React utilizando o ChatGPT como principal ferramenta de apoio.

O estudo foi conduzido de forma interativa, com a IA auxiliando desde a criação de um plano de aprendizado até a explicação de conceitos e exclusão de dúvidas. Cada etapa do aprendizado foi registrada através do Git, adotando a prática de realizar, aproximadamente, um commit por aula. Também foram adicionados commits extras para ajustes, correções e evolução do projeto final. Assim, este repositório não representa apenas um projeto concluído, mas sim todo o caminho percorrido para construí-lo.

## Sobre o Projeto

Esse projeto teve sua aplicação final com a criação do site https://filmslists.vercel.app/, este foi feito inteiramente por mãos humanas, com única exceção do CSS. Para este, utilizei principalmente o Google Gemini e ocasionalmente ChatGPT, pois, além de meu talento para esta arte ser lamentável, o foco do projeto foi aprender React.

O projeto final, entretanto, nunca foi o objetivo, foi apenas o último passo. Durante os vários commits, em que em cada um aprendia uma nova coisa, gerei também outro site, o https://fundamentos-react-ruby.vercel.app/, mas este foi criado passo a passo com o ChatGPT e eu tampouco me importei em embelezá-lo com CSS ou sequer lhe dar um nome.

## Projeto Final

O projeto final, que foi a 16ª aula (a 15ª foi sobre subir sites no Vercel), foi uma sugestão do ChatGPT e consistia em fazer um site que qualquer usuário pudesse acessar um catálogo de filmes por API e favoritar, além de que deveria ter um simulador de login que salvasse no LocalStorage, mas optei por fazer um login com banco de dados real.
